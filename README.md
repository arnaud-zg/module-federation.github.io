## About

A website documenting Webpack 5 Module Federation.

#### *Set-up Locally*
```bash
#install dependencies & run dev server with yarn 
yarn install
yarn dev

#or with npm 
npm install
npm run dev
```
This will start a dev server, navigate to localhost:3000 to check it out.
